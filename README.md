# Hook the Void
Hook the Void is a small physics-based movement sandbox inspired by Nullscape (from Roblox). Swing through a single playground using a grappling hook, build momentum and experiment with movement.

### Features
- Physics-based grappling hook
- Momentum-based movement
- One playground focused on movement
- Hand-crafted visual style

### Controls: 
#### Keyboard & Mouse Only:
- A/D - move left/right
- Left Mouse Click - Attach/Release Grappling Hook

### About
Created in 9 days as a personal challenge to practice finishing projects instead of endlessly polishing them.
